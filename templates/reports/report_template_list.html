About Report Templates

Report templates allow you to define the structure of reports you want to generate regularly. Each template specifies:

  • Entity Type: The type of data to include (beneficiaries, cases, assessments, etc.)
  • Fields: Which fields to include in the report
  • Description: Optional notes about the purpose or content of the report

Once you've created a template, you can use it to generate reports in Excel or PDF format.
